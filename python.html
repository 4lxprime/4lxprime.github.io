Presentation de

python

Presentation globale :

Python est un langage que je présenterais en trois points :

  • 1. simple
  • 2. flexible
  • 3. compatible

Sa simplicité :

python est le langage interprété le plus utilisé au monde.
sa syntaxe est extrêmement simple par rapport à des langages complexes comme le C ou le C++ et il inclut intuitivement les mathématiques.
Voici un exemple de code python qui permet d'afficher un hello world :


python :

print("hello world")


C :

#include <stdio.h>
int main() {
printf("Hello, World!");
return 0;
}

Sa flexibilité :

Python est un langage extrêmement flexible du fait de son fonctionnement avec des modules.
ces modules sont développés pour la grande majorité par sa très grande communauté.
Ils font qu'avec python on peut presque tout faire comme par exemple : des programmes de calculs avancés, des sites web, de la reconnaissance vocale, des assistants vocaux, des intelligences artificielles, des bots discord, de l'automatisme, des malwares ...


Sa compatibilite :

Python est un langage qui est très compatible grace au fait qu'on le retrouve sur des systèmes d'exploitation "normaux" comme linux, windows, macos, android mais aussi sur des machines un peu moins communes comme des systèmes embarqués ou meme des machines quantiques (c'est d'ailleurs le langage priorisé pour les machines quantiques).

Les inconvenients :

Le principal inconvenient de python est sa lenteur, par exemple sur un code équivalent en C et en python, le C va s'exécuter 45 fois plus vite que le python, ce qui est un énorme désavantage.
Mais ce problème peut se régler avec une compilation (avec le programme nuitka) qui va transformer le code en C puis va le compiler pour obtenir un exécutable (.exe), ce qui est l'équivalent d'un code en C compilé.

Pourquoi python ?

Premièrement python est comme je vous l'ai dit précédemment le langage le plus utilisé au monde.
- Secondement python est simple, il a une syntaxe basique, il est très flexible et il est compatible avec de nombreux systèmes d'exploitation.
- Troisièmement python a une très grande communauté et donc vos bénéficierez de plus d'aides disponible sur internet.

Syntaxe basique :

Les variables :

Une variable est une valeur assignée dans le programme.
Il y a plusieurs types de variables :
-les bool (boolean)
-les str (string)
-les int (integer)
...
Les bool servent à définir un état booléen c’est-à-dire True (Vrai) ou False (Faux). Les str servent à mettre des chaines de caractères comme par exemple "hello World!".
Et les int servent à mettre des nombres entiers comme par exemple 562

Les calculs :

Comme je l'ai expliqué précédemment python est un langage qui a une syntaxe orientée vers les mathématiques, donc nous pouvons effectuer des petites opérations avec des + (additions), des - (soustraction), des * (multiplications) ou encore des / (divisions), comme par exemple : 1+5 ou 5 * 8/6 +2

definitions :

- langage interprété = Langage de programmation dont le code source est interprété et non pas compilé. C'est-à-dire que chaque ligne est analysée et exécutée à la volée par un interpréteur.

- hello world = « Hello world » sont les mots traditionnellement écrits par un programme informatique simple dont le but est de faire la démonstration rapide de son exécution sans erreur.

- modules = fichier (dans notre cas en python) qui permet d'importer du code dans son programme sans avoir a le copier coller

- malware = virus informatique

- windows = Système d'exploitation développé par Microsoft

- linux = Système d'exploitation développé par Linus Torvalds qui sert principalement pour les serveurs et la cybersecurité, son kernel est open source

- macos = Système d'exploitation développé par apple pour ses ordinateurs mac

- android = Système d'exploitation développé par google, il utilise le kernel linux et il est utilisé sur la plupart des telephones

- système embarqué = système électronique et informatique autonome dédié à une tâche précise (comme par exemple dans les voitures modernes comme les tesla)

- kernel linux = noyau du systeme d'exploitation linux.

- ordinateur quantique = ordinateur qui effectue des calculs en utilisant directement les lois de la physique quantique

- open source = qui ne cache pas son code et l'expose pour permettre a sa communaute de l'utiliser ou de l'ameliorer.

- IDE = Un environnement de développement intégré
